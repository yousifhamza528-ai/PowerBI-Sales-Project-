# مشروع Power BI - تحليل بيانات المبيعات

## نبذة عن المشروع
هذا المشروع يهدف إلى تحليل بيانات المبيعات باستخدام Power BI.  
يتضمن المشروع تنظيف البيانات، تحليلها، وعرضها بصريًا من خلال لوحات تحكم تفاعلية (Dashboards).

## المهارات المستخدمة
- Excel
- Power BI
- SQL (إذا استخدمت قواعد بيانات)
- Python (إذا استخدمت تحليل إضافي)
- Data Cleaning
- Data Visualization

## مكونات المشروع
1. ملف Power BI (.pbix) يحتوي على الجداول والرسوم البيانية ولوحات التحكم.
2. ملفات البيانات (CSV أو Excel) المستخدمة في المشروع.
3. هذا الملف README لشرح المشروع.

## كيفية استخدام المشروع
1. افتح ملف `.pbix` في Power BI Desktop.
2. استعرض الجداول والرسوم البيانية.
3. يمكنك تعديل البيانات أو إنشاء Dashboard جديد بناءً على المشروع.
